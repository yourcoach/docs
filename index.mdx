This document describes how to integrate and utilize the YourCoach widget within
a partner's ("Company") application. The YourCoach widget is designed as a web
page intended to be embedded via an iFrame into the host platform.

The integration supports two core user flows:

<CardGroup cols={2}>
  <Card title="Signup mode" icon="user-plus">
    Initiates a new user registration flow, processes informed consent, builds
    the program parameters, and triggers automated coach matching.
  </Card>
  <Card title="Login mode" icon="right-to-bracket">
    Directly opens the widget interface for an already enrolled/registered user,
    taking them straight to active coaching program screens and messaging chats.
  </Card>
</CardGroup>

The widget utilizes the HTML5 postMessage API to establish a secure runtime
handshake and transfer the authorization payload from the parent page to the
embedded iFrame context.
