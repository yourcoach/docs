All backend communication uses a JSON-RPC-like POST format.

<Info>
  Each request body must include `id`, `meta`, `method`, and `params`. The
  `method` value identifies the specific API operation (e.g.,
  `user.auth.check_status`).
</Info>

Select the appropriate base URL for your deployment environment:

<Icon icon="flask" /> **Staging / Pre-production**

```bash
https://api.rc.yourcoach.health
```

<Icon icon="rocket" /> **Production**

```bash
https://api.yourcoach.health
```
