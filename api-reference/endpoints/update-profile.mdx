---
title: "Update user profile"
description:
  "user.hooks.update_profile — update demographic information for a user in the
  YourCoach database."
icon: "user-pen"
---

Enables direct demographic updates to the user profile stored on the YourCoach
database.

**Method:** `user.hooks.update_profile`

## Request parameters

<ParamField body="client_id" type="string" required>
  Your company client identifier.
</ParamField>

<ParamField body="client_secret" type="string" required>
  Your company client secret.
</ParamField>

<ParamField body="token" type="string" required>
  Signed authorization JWT for the target user. See [JWT
  generation](/scripts/jwt-generation).
</ParamField>

<ParamField body="demographics" type="object" required>
  Updated demographic fields for the user.

  <Expandable title="demographics fields">
    <ParamField body="first_name" type="string">First name.</ParamField>
    <ParamField body="last_name" type="string">Last name.</ParamField>
    <ParamField body="email" type="string">Email address.</ParamField>
    <ParamField body="phone" type="string | null">Phone number in E.164 format.</ParamField>
    <ParamField body="state" type="string">Two-letter US state code.</ParamField>
    <ParamField body="zipcode" type="string">ZIP code.</ParamField>
    <ParamField body="date_of_birth" type="string | null">Date of birth in `YYYY-MM-DD` format.</ParamField>
    <ParamField body="gender" type="string">Accepted values: `"female"` | `"male"` | `"na"` | `null`</ParamField>
  </Expandable>
</ParamField>
